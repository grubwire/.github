### [Uncorked](https://github.com/grubwire/Uncorked) 'macOS' 'Wine' 'Whisky' 

Run Windows apps and games on your Mac. No Terminal, no scripts, just
download and go.

A fork of Whisky, kept current with Gcenx's Wine builds. Whisky is no
longer maintained; Uncorked carries it forward.
